# 📊 HR Analytics Dashboard | Power BI

> An interactive HR Analytics Dashboard built using **Power BI** to analyze employee demographics, attrition, salary distribution, job satisfaction, and workforce trends. This project demonstrates data visualization, DAX, Power Query, and business intelligence skills.

---

## 📌 Project Overview

The **HR Analytics Dashboard** helps HR professionals and business leaders monitor key workforce metrics and identify patterns in employee attrition and organizational performance.

The dashboard transforms raw HR data into meaningful insights through interactive reports, enabling data-driven decision-making.

## 🎯 Objectives

- Analyze employee attrition.
- Monitor workforce demographics.
- Understand salary distribution.
- Evaluate employee job satisfaction.
- Compare department-wise employee count.
- Identify experience and age trends.

---

## 📈 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| 👥 Total Employees | Total workforce |
| ✅ Active Employees | Currently active employees |
| 📉 Attrition Count | Employees who left |
| 📊 Attrition Rate | Overall attrition percentage |
| 🎂 Average Age | Average employee age |
| 💼 Average Experience | Average years of experience |
